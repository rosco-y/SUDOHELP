SudoKube Options

The options in SudoKube are described as:

  1. Reveal Errors.  When selected, Errors are displayed in Red to indicate that the value is incorrect.  Otherwise incorrect values are presented in black or blue (dependinging on the "Blue User Values" option.
  2. Animate Rotations.  When selected, rotations of the SudoKube are slightly more gradual, making it easier to see which way the Kube was rotated.
  3. Cleanup Candidates.  When a Cell is Solved, all the Candidates with the solution-value effected by the solution are cleared.  These candidates are influenced by the new value.
  4. Click on Values.  Left-Click on a Solution-Value (correct value for a cell.) to clear all the candidates with that value related to the cell with the solution value.  This is a quick way to clear multiple inaccurate candidates.
  5. Right Click on Values.  This option allows you to right-click on any displayed value in the SudoKube (either your solution-values, or values supplied by the puzzle but not candidate-values) to change the Selected Value to that value.  Subsequent left-clicks on empty cells in the SudoKube sets the cube's value to the SelectedValue.  Right-clicks on empty cells to set one of the candidates for the cell to that value.
  6. Highlight Selected Value.  All occurances of the Selected Value in the SudoKube will be Italicized when this option is selected.
  7. Blue Solution Values  Display correct Solution values in Blue.

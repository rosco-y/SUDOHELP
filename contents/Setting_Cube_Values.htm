Setting Cube Values

To solve blank cells in SudoKube, you first set the Selected Value, and then you left-click on the cell to set it's solution.

You First set the "Selected Value" and left-click on Holes in the SudoKu puzzle to set their solutions to that selected value.  There are three ways to set (and de-select) the selected value.

  1) The first way to set the Selected Value is by left-clicking on the numbers in the Selected-Value-Column.

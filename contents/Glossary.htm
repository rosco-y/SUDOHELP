Glossary

There are some terms used throughout the help system that it may be helpful to define:

  1. Layer

  2. Given
    A given is a Sudoku Cell initially presented as a clue to the puzzle.
  3. Region
    A Region is a 3x3 subgrid of a Sudoku puzzle.  When the puzzle is completed, each Region contains the Values 1-9 with no omissions.  There are 9 regions in a Sudoku Puzzle, they are ususally set apart with extra space between the regions.

  4. Holes or Cubes
    The term "Holes" and "Cubes" are used interchangebly in SudoKube.  A hole is a cube, which are placeholders for Values in the Sudoku puzzles which are not givens, and therefore are blank at the outset of the puzzle.
